Freesound Music Automation

Automated music scoring and download system using Freesound.org API

Project Overview

The Freesound Music Automation project is a Python-based tool designed to automatically search, download, and organize background music samples from Freesound.org for use in YouTube videos and other multimedia projects.

This project leverages the Freesound APIv2 to programmatically access a vast library of Creative Commons licensed audio files, enabling automated music curation with intelligent filtering based on duration, ratings, and tags.

How It Works

1. Authentication

The system uses API key authentication stored in environment variables:

FREESOUND_API_KEY=your_api_key_here
Important: You need to obtain your API key from https://freesound.org/apiv2/apply

2. Search Configuration

The downloader searches for music with specific criteria:

  • Duration: 10-60 seconds (ideal for background music)
  • Query: "music background"
  • Filter: Tagged as "music"
  • Sort: By highest rating
  • Results: Top 3 samples per run

3. Download Process

For each sound found:

  • Downloads high-quality MP3 preview
  • Saves to weekly folder (e.g., 3_Simulation/downloads/2026-07/)
  • Creates sanitized filename: freesound_{id}_{name}.mp3
  • Generates JSON metadata file with license and attribution info

4. Metadata Storage

Each download includes a JSON file with complete information:

{ "id": 12345, "name": "Background Music Loop", "duration": 30.5, "username": "AudioCreator", "license": "Creative Commons 0", "url": "https://freesound.org/people/...", "filename": "freesound_12345_Background_Music_Loop.mp3" }

Getting Started

Prerequisites

  • Python 3.14 or higher
  • Freesound API key
  • Virtual environment (recommended)

Installation

# Clone the repository git clone <repository-url> # Navigate to project directory cd Freesound # Create virtual environment python3 -m venv venv # Activate virtual environment source venv/bin/activate # On Windows: venv\Scripts\activate # Install dependencies pip install requests python-dotenv # Create .env file echo "FREESOUND_API_KEY=your_api_key_here" > .env

Usage

# Run the downloader python 5_Symbols/freesound_downloader.py # Output will be saved to: # 3_Simulation/downloads/YYYY-WW/

API Features

The Freesound APIv2 provides extensive capabilities:

  • Text Search: Search by keywords, tags, and descriptions
  • Content Search: Filter by audio features (pitch, BPM, etc.)
  • Combined Search: Mix textual and content-based queries
  • Sound Download: Access original files or optimized previews
  • Upload Capability: Upload your own sounds (OAuth2 required)
  • User & Pack Info: Retrieve detailed metadata
  • Similar Sounds: Find sounds similar to a reference

Rate Limits

  • Standard: 60 requests/minute, 2000 requests/day
  • Write operations: 30 requests/minute, 500 requests/day

License Compliance

All downloaded sounds from Freesound are released under Creative Commons licenses. The system automatically tracks license information for each download, ensuring proper attribution when using the sounds in your projects.

Always check the .json metadata files for license details and required attribution before using sounds in commercial or public projects.
